# Kouban-Lost-Found-
User authentication
- Auth0 (probably best)
- Cloud (Redis or SSO)
- Firebase

Register Lost Item
- View lost items in specific wards
- Can enter radius of where item was last seen(google maps)
- Printable lost item confirmation slip 

View updates of items previously entered

- Can edit previous entry
